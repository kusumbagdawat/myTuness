# 🎵 My Tunes - Frontend Music Website

A modern and responsive music website frontend built using **HTML5**, **CSS3**, **Tailwind CSS**, and **Bootstrap**. This project showcases a clean user interface for a music streaming platform with sections like Home, Features, Artists, Albums, and Contact.

## 🚀 Features

- 🎧 Responsive design for all screen sizes
- 🎨 Modern and attractive UI
- 📱 Mobile-friendly layout
- 🎵 Music-themed landing page
- 📂 Organized and reusable code structure
- ⚡ Built with Tailwind CSS and Bootstrap components
- 🖱️ Smooth navigation and interactive elements

## 🛠️ Technologies Used

- HTML5
- CSS3
- Tailwind CSS
- Bootstrap 5
